Configuration base de donnees

Pour autoriser un compte, cree ou complete son document dans clients/{uid} avec le role admin.

  • Etape 1. Cree le compte admin dans Firebase Authentication.
  • Etape 2. Dans Firestore, cree ou mets a jour clients/{uid}.
  • Etape 3. Ajoute au minimum role: "admin" et idealement dashboardAccess: true.
  • Etape 4. Une fois connecte avec ce compte, le dashboard s ouvrira automatiquement.
