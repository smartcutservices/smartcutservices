Bootstrap admin

Creer le compte admin du dashboard.

Cette page initialise automatiquement le compte admin dans Firebase Authentication et dans Firestore, sans creation manuelle dans la base de donnees.

  • 1. Le compte est cree dans Firebase Authentication.
  • 2. Le document clients/{uid} est cree avec role: "admin".
  • 3. Le dashboard pourra ensuite verifier ce role automatiquement.
